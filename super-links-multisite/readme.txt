== Description ==

O Super Links permite que você crie links usando seu próprio nome de domínio que redirecionam para seus links de afiliado de forma camuflada ou não. Você pode inserir códigos de rastreio do Facebook e Google de forma simples e fácil. E ainda monitorar a quantidade acessos que cada link obteve.
Além disso, ainda é possível camuflar seus links para que usuários de determinados países sejam redirecionados para seus links de afiliados, enquanto robôs e outros acessos (Facebook e Google por exemplo) visualizem outra página que você configurar.

== Installation ==

1. Envie o arquivo 'super-links.zip' para o diretório '/wp-content/plugins/'

2. Ative o plugin através do menu 'Plugins' em seu painel administrativo do WordPress

3. Certifique-se de ter alterado as Configurações de links permanentes no menu 'Configurações' do seu painel WordPress para qualquer configuração diferente da padrão
